ХИРУРГИЧЕСКАЯ ОПЕРАЦИЯ: ОПРЕДЕЛЕНИЕ, КЛАССИФИКАЦИЯ ОПЕРАЦИЙ ПО ЦЕЛИ ВМЕШАТЕЛЬСТВА; ПО СРОЧНОСТИ; ПО КОЛИЧЕСТВУ ЭТАПОВ; ПО ПЛАНИРУЕМОМУ РЕЗУЛЬТАТУ; ПО ТЕХНИКЕ ВЫПОЛНЕНИЯ; СПЕЦИАЛЬНЫЕ ВИДЫ ОПЕРАЦИЙ.

Хирургическая операция – медицинская манипуляция, выполняемая по установленным правилам с помощью различных способов разъединения, перемещения и соединения тканей.

Классификация операций:

По цели:

Лечебные

Диагностические

По срочности:

Экстренные (при внезапном возникновении заболевания, представляющего угрозу для жизни пациента)

Неотложные или срочные (то же, только без угрозы жизни)

Плановые

По количеству этапов:

Одномоментные

Двухмоментные

Многомоментные

По планируемому результату:

Радикальные (полное излечение)

Паллиативные (облегчение состояния, улучшение качества жизни)

По технике выполнения:

Открытые (классические или традиционные)

Малоинвазивные (лапароскопические, эндоскопические в то числе NOTES технологии (natural orifice translumenal endoscopic surgery)

Микрохирургические

Специальные виды операций:

Пластические

Реконструктивные

Повторные

Высокотехнологические

Вверх
К оглавлению
